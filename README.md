UI-INPUT-ICON
===========================

Icon input element for polymer web components

# Installation

##bower

``` bash

bower install ui-input-icon

```

## Usage

```html

<link rel="import" href="bower_components/ui-input-icon/ui-input-icon.html">


```

## Element Attributes

``` bash
data-id
data-name
placeholder
role
icon : Html Entity representation, e.g, &#xe609;

```
